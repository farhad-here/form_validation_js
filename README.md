# 🎱Form Validation 
#### 📓 This project is a simple responsive JavaScript form validation example.

### 🗒️The form checks the following:

- Features
  - Username Validation
    Must be at least 5 characters long
    If not valid, an error message is displayed:
    “Username must be at least 5 characters long”

  - Password Validation
    Must contain:
      1) at least one uppercase letter
      2) at least one lowercase letter
      3) at least one number
    If invalid, the following error message appears:
    “Password must contain at least one uppercase letter, one lowercase letter, and one number.”

- Successful Submission
  If both fields pass validation, the message displayed is:
  “Form submitted successfully!”

### ❔ How It Works
  The form listens for the submit event.
  The input values are checked using:
  Length validation for username
  Regular expressions (RegEx) for password
  Appropriate messages are shown based on the validation results.
